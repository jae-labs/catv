# Security Policy

## Reporting Security Issues
If you believe you have found a security vulnerability in this project, please report it directly through the GitHub Security Advisory feature.

- Please do **not** create public issues for security vulnerabilities.
- We will respond as quickly as possible to your report.
- Once the issue is resolved, we will disclose the details responsibly.

## Supported Versions
Only the latest major release is actively supported. Older versions may not receive security updates.

## Dependency Management
This project uses GitHub Dependabot to monitor and update dependencies for known vulnerabilities.
