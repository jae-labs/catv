# Contributing to catv

Thank you for your interest in contributing! Please follow these guidelines to help us maintain a welcoming and productive environment.


## How to Contribute
- Before starting work, please open an issue describing your proposed change or feature.
- Discuss your idea with maintainers and wait for feedback.
- Fork the repository and create your branch from `main`.
- Make your changes and submit a pull request (PR).
- Link your PR to the relevant issue.
- Ensure your code follows our style and passes all tests.

## Reporting Issues
- Use GitHub Issues to report bugs or request features.
- Provide as much detail as possible.

## Pull Request Process
- Describe your changes clearly.
- Link to any relevant issues.
- Ensure CI checks pass before requesting review.

## Code Review
- All submissions are reviewed by maintainers.
- Please be responsive to feedback and make requested changes.

## Community
- Be respectful and constructive in all communications.
